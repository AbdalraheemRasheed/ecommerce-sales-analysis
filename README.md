## 🚀 How to Run
1. Clone the repository
2. Install requirements: `pip install pandas`
3. Open `analysis.ipynb` to view the Python analysis and data cleaning
4. Open `visuals/charts.pdf` or the Excel file to explore the pivot tables and charts
