# E-Commerce Sales Analysis

Sales performance analysis for e-commerce data: revenue trends, top products, and customer behavior insights using Python and Excel.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Excel
- Jupyter Notebook

## 📊 Analysis Includes
- Revenue trends over time
- Top performing products
- Customer behavior insights
- Data cleaning & preparation

## 📁 Project Structure
- `analysis.ipynb` — Main Python notebook
- `data/` — Dataset files
- `visuals/` — Charts and graphs

## 🚀 How to Run
1. Clone the repository
2. Install requirements: `pip install pandas matplotlib`
3. Open `analysis.ipynb` in Jupyter Notebook
